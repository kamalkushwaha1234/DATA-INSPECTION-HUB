Categorical variables: These are variables that represent categories or groups, such as gender, ethnicity, occupation, or product type.
Boolean variables: These are variables that take on one of two possible values, such as "yes" or "no", "true" or "false", or 1 or 0.
Ordinal variables: These are categorical variables that have a natural ordering or ranking, such as education level (e.g., high school, college, graduate school) or income level (e.g., low, medium, high).
Continuous variables: These are numeric variables that can take on an infinite number of values within a certain range, such as time, distance, or temperature.
Discrete variables: These are numeric variables that can only take on a finite number of values within a certain range, such as the number of children in a family or the number of products sold by a company in a day.

Effect of duplicate value in dataset

Advantages:
Increased sample size: Duplicate records can increase the sample size of the dataset, which can be beneficial for training machine learning models.
Increased accuracy: If the dataset contains duplicate records, the model can use this to weigh certain records more heavily, which can lead to increased accuracy in predictions.

Disadvantages: Biased results: Duplicate records can skew the results of the analysis, leading to biased or inaccurate predictions.
Reduced performance: Duplicate records can make the training process longer, reducing the performance of machine learning models.
Overfitting: Duplicate records can lead to overfitting, where the model becomes too reliant on specific data points in the dataset and performs
poorly when presented with new data.

Overall, while duplicate records can increase the sample size and accuracy of a dataset, it is generally best to avoid them, as they can introduce biases and negatively impact the performance of machine learning models.


Some Information About NUMERIC Features

count: The number of non-null values in each column. If a column has missing values,
the count value will be lower than the total number of rows in the DataFrame.
mean: The average value of each column.
std: The standard deviation of each column. It measures how much the values in a column deviate from the mean.
min: The minimum value of each column.
25%, 50%, 75%: The quartile values of each column. The quartiles divide the values in a column into four equal parts.
The 25th percentile is the value below which 25% of the observations fall, the 50th percentile is the median
or the value below which 50% of the observations fall and the 75th percentile is the value below which 75% of
the observations fall.
max: The maximum value of each column.

Skewness and Kurtosis

A normal distribution should have a skewness value of zero and a kurtosis value of 3.
If the skewness is negative, the distribution is skewed to the left, and if the skewness is positive, the distribution is skewed to the right.
If the kurtosis is less than 3, the distribution is platykurtic (flatter than normal), and if it is greater than 3, it is leptokurtic (more peaked than normal).
